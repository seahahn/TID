# TID

> Today I Did : Recording not only what I learned, also what I experienced

매일 배운 것뿐만 아니라 트러블 슈팅(속칭 삽질), 요동치는 심리의 변화 등 개발 도중의 경험 모두를 기록합니다.

## Directory Structure

- DailyRecord
  - 일일 기록을 남깁니다.
  - 날 것 그대로인지라 형식같은 것은 adios.
  - 주요 내용
    - 수행한 작업
    - 트러블 슈팅
    - 학습 내용
      <br/>
- StudyRecord
  - 학습한 내용에 대하여 각 잡고 기록을 남깁니다.
  - 블로그에 업로드한다는 자세로 내용 정리를 합니다.
    <br/>
- QriosityRecord
  - 뭐든 하다가 궁금한 개념 등이 있으면 키워드를 기록해둡니다.
  - Daily 또는 StudyRecord에 이에 대한 학습 내용을 기록합니다.
  - 기록이 완료되면 QriosityRecord에 해당 기록 링크를 남깁니다.
