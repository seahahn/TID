# Image filter 속성

- 이미지에 필터를 걸기 위한 속성
- png 등의 비트맵 이미지에는 바로 적용됨
- svg의 경우 이미지 자체에 gradient가 적용된 경우 filter 속성이 제대로 적용되지 않음
  - gradient가 없으면 비트맵 이미지와 동일한 방식으로 적용됨
